关于 Kali Linux 和 Kali Nethunter

Kali Linux 是一个基于 Debian 的 Linux 发行版，专为数字取证和渗透测试而设计。它包含了数百种安全工具，是安全专业人员的标准工具集。

Kali Nethunter 是 Kali Linux 的移动版本，可以在 Android 设备上运行，让安全专业人员和爱好者能够在移动设备上执行安全测试。

重要提示： Kali Nethunter 可以在手机上使用，但请注意，它需要手机的 root 权限 才能充分发挥功能。

Kali Linux 安装教程

准备工作

在安装 Kali Linux 之前，请确保您有以下条件：

至少 20GB 的可用磁盘空间

至少 2GB 的 RAM（推荐 4GB 或更多）

能够从 USB 启动的计算机

一个 8GB 或更大的 USB 驱动器

安装步骤

  1. 从 Kali Linux 官网下载最新的 ISO 镜像文件
  2. 使用 Rufus 或类似工具将 ISO 写入 USB 驱动器
  3. 重启计算机并从 USB 驱动器启动
  4. 选择 "Graphical install" 选项
  5. 按照屏幕上的提示完成安装过程
  6. 设置用户名、密码和时区
  7. 等待安装完成并重启系统
# 安装后更新系统
sudo apt update
sudo apt full-upgrade -y

# 安装常用工具
sudo apt install -y kali-tools-top10

注意： Kali Linux 是一个强大的安全工具，请仅在您拥有权限的系统上使用它。

Kali Nethunter 安装教程

准备工作

在安装 Kali Nethunter 之前，请确保您有以下条件：

已 root 的 Android 设备

足够的存储空间（至少 5GB）

稳定的互联网连接

已安装 Termux 应用

警告： Kali Nethunter 需要手机的 root 权限。对设备进行 root 操作可能会使您的设备保修失效，并存在安全风险。请谨慎操作。

安装步骤

  1. 在 Android 设备上安装 Termux 应用
  2. 打开 Termux 并更新包管理器：
pkg update && pkg upgrade -y
  1. 安装必要的依赖：
pkg install -y wget proot
  1. 下载并运行 Kali Nethunter 安装脚本：
wget -O install-nethunter-termux https://offs.ec/2MceZWr
chmod +x install-nethunter-termux
./install-nethunter-termux
  1. 等待安装完成，然后启动 Kali Nethunter：
nh
# 或使用完整命令
nethunter
  1. 设置 Kali Nethunter 密码：
passwd

提示： 安装完成后，您可以使用 nethunter 命令启动 Kali 环境，使用 nethunter kex 启动图形界面。

Kali Linux 中文输入法安装教程

输入法框架选择

Kali Linux 支持多种中文输入法框架，最常用的是 Fcitx 和 IBus。以下是两种框架的对比：

Fcitx 框架

Fcitx 是一个轻量级且功能强大的输入法框架，在 Kali Linux 中广泛使用。

优点
  • 轻量级，资源占用少
  • 支持多种输入法
  • 配置灵活
缺点
  • 配置相对复杂
  • 需要手动设置环境变量

IBus 框架

IBus 是 GNOME 桌面环境的默认输入法框架，集成度较高。

优点
  • 与 GNOME 集成良好
  • 配置简单直观
  • 支持多种语言
缺点
  • 资源占用相对较高
  • 在某些应用中兼容性可能有问题

Fcitx 输入法框架安装步骤

  1. 更新软件包列表：
sudo apt update
  1. 安装 Fcitx 框架和谷歌拼音输入法：
sudo apt install fcitx fcitx-googlepinyin fcitx-config-gtk
  1. 配置输入法系统：
im-config

在打开的界面中选择 fcitx 作为默认输入法框架。

  1. 设置环境变量，编辑 ~/.bashrc 文件：
vim ~/.bashrc

在文件末尾添加以下内容：

export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx
  1. 使配置生效：
source ~/.bashrc
  1. 重启系统：
sudo reboot
  1. 重启后，打开 Fcitx 配置工具添加中文输入法：
fcitx-configtool

点击 "+" 按钮，取消勾选 "Only Show Current Language"，搜索并添加 "Google Pinyin"。

提示： 安装完成后，可以使用 Ctrl+Space 切换中英文输入法。

IBus 输入法框架安装步骤

  1. 更新软件包列表：
sudo apt update
  1. 安装 IBus 框架和拼音输入法：
sudo apt install ibus ibus-pinyin
  1. 启动 IBus 设置：
ibus-setup
  1. 在打开的界面中，选择 "Input Method" 标签页，点击 "Add" 按钮
  2. 选择 "Chinese" -> "Pinyin" 输入法
  3. 配置输入法系统：
im-config

在打开的界面中选择 ibus 作为默认输入法框架。

  1. 设置环境变量，编辑 /etc/environment 文件：
sudo vim /etc/environment

在文件末尾添加以下内容：

LANG="zh_CN.UTF-8"
UBUNTU_MIRROR="http://cn.archive.ubuntu.com/ubuntu/"
LC_ALL="zh_CN.UTF-8"
  1. 重启 IBus 服务：
ibus restart
  1. 重启系统使所有更改生效：
sudo reboot

提示： 安装完成后，可以使用 Super+Space 切换中英文输入法。

Kali Nethunter 中文输入法安装

在 Kali Nethunter 中安装中文输入法与在标准 Kali Linux 中类似，但需要注意以下几点：

  1. 确保 Nethunter 系统已更新：
sudo apt update && sudo apt upgrade -y
  1. 安装 IBus 和拼音输入法（推荐在 Nethunter 中使用）：
sudo apt install ibus ibus-pinyin
  1. 启动 IBus 守护进程：
ibus-daemon -drx
  1. 打开 IBus 首选项：
ibus-setup
  1. 在 IBus 首选项中添加中文输入法：

进入 "Input Method" 标签页，点击 "Add" 按钮，选择 "Chinese" -> "Pinyin"。

  1. 设置 IBus 开机自启动：

编辑 ~/.bashrc 文件，在末尾添加：

export GTK_IM_MODULE=ibus
export XMODIFIERS=@im=ibus
export QT_IM_MODULE=ibus
ibus-daemon -drx
  1. 使配置生效：
source ~/.bashrc

注意： 在 Kali Nethunter 中，由于是运行在移动设备上，可能需要额外的配置才能在某些应用中正常使用中文输入法。

常见问题解决

中文显示乱码问题： 如果遇到中文显示乱码，可以安装中文字体包：

sudo apt install fonts-noto-cjk ttf-wqy-microhei ttf-wqy-zenhei

输入法无法启动： 如果输入法无法正常启动，尝试重启输入法框架：

# 对于 Fcitx
fcitx -r

# 对于 IBus
ibus restart

输入法切换快捷键无效： 如果快捷键无效，检查系统键盘快捷键设置，确保没有冲突。
